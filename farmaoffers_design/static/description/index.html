FarmaOffers Module

Core backend models, views, and controllers for FarmaOffers.

Overview

FarmaOffers Module provides the core backend structure used by the FarmaOffers platform. It includes administrative models, business logic, controllers, and backend views required to manage website content and user submissions.

Main Features

  • Backend models for prescriptions, quotes, contact requests, FAQs, tips, banners, and sliders
  • Administrative list and form views for managing user-submitted data
  • Controllers handling public website forms and data persistence
  • Menu entries and actions integrated into the Odoo backend
  • Reusable backend logic supporting FarmaOffers website flows

Technical Scope

This module is intended to be installed as a backend dependency. It does not provide a standalone theme, but rather supports the FarmaOffers website and sales ecosystem by exposing structured backend components.
